# PyCalc-Manager

### Mi primer proyecto integral en Python
Una herramienta dual que combina **lógica matemática** y **gestión de gastos**. 

El objetivo de este prototipo es aplicar conceptos fundamentales:
* **Bucles infinitos** para menús interactivos.
* **Validación de entradas** de usuario.
* **Formato de datos financieros** (f-strings).
* **Persistencia de datos** en archivos locales (.txt).
